Added Meta Description

Semantic HTML Added

<header>...</header>
<nav aria-label="Main Navigation">...</nav>
<main>...</main>
<footer>...</footer>

ALT Text Added

<img src="profile.jpg" alt="Profile picture of Abu Bakar">

ARIA Roles Added

3. Accessibility Fixes

  • Improved heading structure (H1 → H2 → H3)
  • Added keyboard focus styles
  • Improved color contrast
  • Added labels to forms
  • Made all buttons accessible
  • Removed empty links

Keyboard Focus CSS

  outline: 3px solid #005bcc;
